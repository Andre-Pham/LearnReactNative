```

```

#### Running on Android

First have an Android emulator open.

```
/Users/andrepham/Library/Android/sdk/emulator/emulator -avd Pixel_6_Pro_API_33 -netdelay none -netspeed full
```

Then run.

```
npx react-native run-android
```
